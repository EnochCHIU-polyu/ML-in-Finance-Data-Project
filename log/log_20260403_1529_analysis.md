### 3. System Improvement Recommendations (Deep Analysis)

#### Focus 1: Cluster News Method
- **Current Issue**: Lag/decay grid search yields identical MAE across all (lookback, decay) combinations (all return 4.8484), suggesting the grid search validation set may be too small or the sentiment signal is already being fully absorbed at any lag depth.
- **Recommendation A**: Switch to walk-forward cross-validation (TimeSeriesSplit with 5 folds) for grid search instead of single val set.
- **Recommendation B**: Use TF-IDF + BM25 pre-filtering before FinBERT to reduce irrelevant headlines.
- **Recommendation C**: Add sector-aware sentiment weighting: weight peer-stock sentiment by return correlation coefficient, not equal-weight.
- **Recommendation D**: Introduce sentiment regime detection — only activate cluster-news features when VIX > 20 (high uncertainty periods where news matters most).
- **Recommendation E**: Replace K-Means (Euclidean) clustering of stocks with correlation-based affinity clustering (e.g., OPTICS on correlation distance matrix) for more financially meaningful peer groups.
